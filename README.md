# Yandex Maps Fuel Cost Calculator

Расширение для Chrome и Яндекс Браузера, которое показывает стоимость бензина для маршрутов на Yandex Maps (список вариантов маршрута).

## Возможности
- Автоматический расчёт стоимости топлива для каждого маршрута.
- Если Яндекс показывает цену платной дороги в карточке маршрута, она прибавляется к итоговой цене.
- Отображение рядом с расстоянием в списке маршрутов (оранжевый бейдж).
- Настройки цены топлива и расхода прямо на странице (кнопка ⛽).
- Значения сохраняются локально.

## Установка (распакованная)
1. Скачайте или клонируйте репозиторий.
2. Откройте `chrome://extensions/` и включите «Режим разработчика».
3. Нажмите «Загрузить распакованное расширение» и укажите папку проекта.

## Использование
1. Откройте [Yandex Maps](https://yandex.ru/maps/) и постройте маршрут.
2. Нажмите кнопку ⛽ в правом верхнем углу, укажите:
   - Цена топлива (₽/литр)
   - Расход топлива (л/100 км)
3. Нажмите «Применить» — стоимость появится рядом с расстоянием маршрута.

## Формула
```
Стоимость = (Расстояние в км × Расход л/100км / 100) × Цена за литр
```

Если для маршрута показана платная дорога, итоговая цена считается как:

```
Итог = Стоимость топлива + Цена платной дороги
```

## Технические детали
- Manifest V3.
- Работает только на `https://yandex.ru/maps/*`.
- Настройки хранятся в `chrome.storage.local`.
- Фильтры исключают элементы масштаба, поиск, панель настроек.

## Структура
- `AGENTS.md` — краткие правила для будущих агентов и автоматизаций.
- `manifest.json` — конфигурация расширения.
- `content.js` — логика расчёта и вставки бейджей.
- `fuel-core.js` — pure helpers для парсинга расстояния и расчёта стоимости.
- `styles.css` — базовые стили бейджей и UI.
- `tests/fuel-core.test.js` — минимальные автопроверки на `node --test`.

## Релизы
- GitHub Release по тегу `v*` собирает zip только из файлов расширения.
- Документация, тесты и служебные файлы в release-архив не попадают.

## Требования
- Chrome 88+ (Manifest V3).
- Иконки опциональны; при отсутствии используйте заглушки или уберите секцию `icons`.

## Разработка
- Запуск автопроверок: `npm test`
- Проверка синтаксиса: `node --check fuel-core.js` и `node --check content.js`
- Проверка diff на whitespace/merge-мусор: `git diff --check`
- Для DOM-отладки можно временно включить `DEBUG_DOM_BREAKAGE = true` в `content.js` и перезагрузить распакованное расширение.
- Короткие маршруты в метрах (`850 м`) поддерживаются так же, как и маршруты в километрах.
